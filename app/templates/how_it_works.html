The book titles and reviews were scrapped from audible.com

The word2vec pretrained model ws downloaded from the google code archive.

Each word in the string is first embbeded to a vector of length = 300, then feed into the recurrent neural network (RNN) model.

The RNN model consists two long-short-term memory (LSTM) layers.

Word embedding

Word embedding allows the model to "understand" word similarity, and this ability can be extended to sentences.

For example:

(If we teach the model)

"it is an amazing book" --> 👍

"blood is everywhere" --> 😱

(The model can do, even when the word "gore" and "awesome" were not included in the training examples.)

"this is awesome" --> 👍

"gore to the max" --> 😱

In other words, the model can understand a sentence even when the model has not "seen" some words in that sentence before.

RNN

An RNN model is capable of "interpreting" sentences beyond the words. So it does not tag a thumb up anytime it sees the word "good", smile face anytime it sees the word "funny".

Examples:

"good" --> 👍

"not good" --> 😒

"this is not funny at all" --> " " (no emoji)

LSTM

LSTM architecture allows the RNN model to "hold that memory" longer, so the model is capable of task like these:

"not at all as good as expected" --> 😒

😀 - 'Plenty of laugh out loud moments'
🤔 - 'This story has been so inspirational to me, and makes you think deeply about current issues'
😥 - 'I wasn't sure what to expect but it left me in tears'
😱 - 'The horror does not disappoint in any of these regards'
😒 - 'I found this book pretty blah'
👍 -'It is an absolute classic for many good reasons'
'no emoji'- 'There are a lot of pros and cons to this book'

Sentiment analysis use cases are usually very context dependent, you probably don't need to identify a "screaming in fear" face in every of your business, but the small training example requirement means that the technique used in this app can easily be transferred to build a customized model.

For example, you can build a model to dedicate customer emails to different departments of the company.

Or, quickly filter out the most urgent comments to a specialized "first aid" group.
