The fuzzy search app using word-embedding

When we search for something, we use keywords. However, we can mess up with the keywords.

With the word-embedding model, we can search by the meaning of the words, rather than depend strictly matching of keywords.

The present app returns results base on the similarity in the meanings between the search keywords and the target item. It's like the app says "OK... I got the idea".

Currently, the app is based on titles of the 1000 best-selling Fiction books listed on audible.com. User can type find A Clockwork Orange by typing in "mechanical tangerine" as the keyword. (This task is currently not as easy on audible.com.)

The emoji tagger

When we read online reviews, we try to interpret how other people feel about something base one the text they typed in. Here, we can instead get a quick emoji summary with a click.

This emoji tagger is base on a Recurrent neural network model, that also empowered by word embedding. We are basically doing sentiment analysis here, but the power of word embedding is that we can train a reasonable model with only thousands of example, this is a very small number for a deep learning model.
