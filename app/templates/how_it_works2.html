The emoji model

Each word in the string is first embbeded to a vector of length = 300, then feed into the recurrent neural network (RNN) model.

The RNN model consists two long-short-term memory (LSTM) layers.

Examples of customized sentiment analysis tasks

Sentiment analysis use cases are usually very context dependent, you probably don't need to identify a "screaming in fear" face in every of your business, but the small training example requirement means that the technique used in this app can easily be transferred to build a customized model.

For example, you can build a model to dedicate customer emails to different departments of the company.

Or, quickly filter out the most urgent comments to a specialized "first aid" group.

RNN

An RNN model is capable of "interpreting" sentences beyond the words. So it does not tag a thumb up anytime it sees the word "good", smile face anytime it sees the word "funny".
